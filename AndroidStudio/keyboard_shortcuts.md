# keyboard shortcuts

* reformat code: `Ctrl` + `Alt` + `L`

* close active editor tab: `Ctrl` + `F4`

* project pane open/close (開/關檔案結構面板): `Alt` + `1`
    * 注意如果游標在程式碼中閃爍的話，快捷鍵可能失效，或是需要多按幾下。

* terminal open/close: 
    * `Alt` + `F12` 
    * `Shift` + `Esc`

* close the whole editor: `Alt` + `F4`
